Instructions to run the project :
==================================
1.	Navigate to the project directory, execute cd twitter/
2.	Fetch all the dependencies, execute “mix deps.get”
3.	Start the phoenix twitter server, execute “mix phx.server” or “iex -S mix phx.server” (for interactive mode)
	Expected output: Running TwitterWeb.Endpoint with Cowboy using http://0.0.0.0:4000
4.	Open a web browser and type in the url “http://0.0.0.0:4000” and hit enter
5.	Follow instructions on the screen to navigate through the application
